# Start automating

Once the server and extension are set up, ask your AI client to call tools:

- `open-tab` – `{ "url": "https://example.com", "active": true }`
- `tab-click` – `{ "selector": "#submit" }`
- `tab-back`, `tab-forward`
- `snapshot` – `{ "url": "https://example.com" }`

Try locally with the MCP Inspector:

```bash
cd server
npm run build
npm run inspect
```
