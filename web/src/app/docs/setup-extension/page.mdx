# Set up extension

1. Open `chrome://extensions` and enable Developer mode.
2. Click “Load unpacked” and select the `extension` folder in this repo.
3. Ensure the HTTP/WS bridge is running at `http://localhost:3000`.
4. Optionally verify status at `http://localhost:3000/ws-status`.
