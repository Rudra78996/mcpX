# MCPX Documentation

MCPX is a powerful browser automation tool that connects AI applications to your browser through the Model Context Protocol (MCP).

## What is MCPX?

MCPX enables AI applications to interact with web browsers for automation, testing, and data extraction tasks. It provides a secure and efficient way to:

- **Automate browser interactions** - Click, type, navigate, and extract data
- **Test web applications** - Run automated tests and quality assurance
- **Extract web data** - Scrape and process information from websites
- **Integrate with AI tools** - Connect browser capabilities to AI workflows
