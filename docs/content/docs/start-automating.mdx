## Start Automating

Once the MCP server and extension are set up, you can start using **Browser MCP** to automate your browser.

### Run a Command

To test the extension, open your MCP client (e.g., Cursor) and run:

```bash
Open youtube.com and search for "AI tutorials"
```

### More Examples

You can try other commands, such as:

```bash
Navigate to github.com and open Rudra78996
Scroll down and take a screenshot
Fill the login form on example.com with test credentials
```
