## Installation

Get started with Browser MCPX in just a few steps.

### Quick Start

1. **Clone the repository**

   ```bash
   git clone https://github.com/Rudra78996/mcpX.git
   cd mcpX
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Build the project**
   ```bash
   npm run build
   ```
