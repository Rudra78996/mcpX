# Installation

This guide will walk you through installing all MCPX components on your system.

## Prerequisites

Before you begin, ensure you have:

- **Node.js** 18 or later
- **npm** or **yarn** package manager
- A compatible browser (Chrome, Firefox, or Edge)

## Step 1: Install Node.js

If you don't have Node.js installed:

### Windows

1. Download from [nodejs.org](https://nodejs.org)
2. Run the installer with default settings
3. Verify installation:
   ```cmd
   node --version
   npm --version
   ```

### macOS

Using Homebrew (recommended):

```bash
brew install node
```

### Linux

```bash
# Ubuntu/Debian
curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash -
sudo apt-get install -y nodejs

# CentOS/RHEL
curl -fsSL https://rpm.nodesource.com/setup_lts.x | sudo bash -
sudo yum install -y nodejs
```

## Step 2: Install MCP Server

### Global Installation (Recommended)

```bash
npm install -g mcpx-server
```

### Local Installation

```bash
mkdir my-mcpx-project
cd my-mcpx-project
npm init -y
npm install mcpx-server
```

### Verify Installation

```bash
mcpx-server --version
mcpx-server --help
```

## Step 3: Install Browser Extension

### Chrome/Edge

1. Visit the [Chrome Web Store](https://chrome.google.com/webstore)
2. Search for "MCPX Browser Automation"
3. Click "Add to Chrome"
4. Confirm permissions

**Manual Installation:**

1. Download from [GitHub Releases](https://github.com/Rudra78996/mcpX/releases)
2. Open `chrome://extensions/`
3. Enable "Developer mode"
4. Click "Load unpacked" and select the extension folder

### Firefox

1. Visit [Firefox Add-ons](https://addons.mozilla.org)
2. Search for "MCPX"
3. Click "Add to Firefox"

**Manual Installation:**

1. Download the `.xpi` file from GitHub
2. Open `about:addons`
3. Click gear icon â†’ "Install Add-on From File"

## Step 4: Configure AI Application

### Claude Desktop

Add to your Claude configuration:

```json
{
  "mcpServers": {
    "mcpx": {
      "command": "mcpx-server",
      "args": ["--port", "3000"]
    }
  }
}
```

### Other MCP Clients

For other MCP-compatible applications, add similar configuration pointing to `mcpx-server`.

## Verification

Test your installation:

1. **Start the server:**

   ```bash
   mcpx-server --port 3000
   ```

2. **Check extension:** Look for the MCPX icon in your browser toolbar

3. **Test connection:** In your AI application, try: "Open google.com"

## Troubleshooting Installation

### Common Issues

**"command not found: mcpx-server"**

- Ensure Node.js is properly installed
- Try reinstalling: `npm install -g mcpx-server`
- Check your PATH environment variable

**Extension not loading**

- Verify browser compatibility
- Enable Developer mode for manual installs
- Check browser console for errors

**Permission errors**

- Run terminal as administrator (Windows)
- Use `sudo` for global installs (macOS/Linux)
- Check file permissions
